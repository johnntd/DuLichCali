SMS Opt-In Disclosure

Last updated: May 5, 2026  ·  Du Lịch Cali

DuLichCali sends transactional SMS messages only to users who request a booking, food order, airport pickup/ride, or service inquiry on our website.

How users opt in

  1. User visits https://www.dulichcali21.com
  2. User selects a service such as salon booking, food order, airport pickup/ride request, or service inquiry
  3. User enters their name, mobile phone number, and request details
  4. Before submitting, user must actively check an unchecked SMS consent checkbox
  5. User then clicks Confirm Booking, Place Order, Request Ride, or Submit Request
  6. SMS is sent only for updates related to that user's own request

Consent checkbox

The following checkbox appears unchecked on every booking, order, and ride request form. Users must actively check it before submitting:

Above: the checkbox as it appears to users — unchecked by default. Submission is blocked until the user actively checks it.

Message types

All SMS messages sent through this campaign are transactional:

  • Appointment confirmations
  • Booking updates
  • Appointment reminders
  • Food order confirmations
  • Order ready notices
  • Airport pickup or ride updates
  • Customer support replies related to the user's request

No marketing SMS

DuLichCali does not use this SMS campaign for marketing, advertising, promotions, abandoned-cart messages, or unsolicited outreach.

Opt out and help

Users can reply STOP to unsubscribe or HELP for help. Message frequency varies. Message and data rates may apply.
