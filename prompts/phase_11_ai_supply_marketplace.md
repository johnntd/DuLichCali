# Phase 11 — Nail Supply Marketplace Integration

Goal:
Turn restocking into a marketplace opportunity inside DuLichCali.

Features:
- Vendor can browse supply recommendations
- Supplies grouped by category
- AI compares price, quality, shipping, and vendor preference
- Vendors can save preferred suppliers
- Vendors can mark item as purchased
- System updates inventory after restock confirmation

Marketplace categories:
- Gel polish
- Acrylic
- Dip powder
- Nail tips
- Tools
- Disposable supplies
- Salon cleaning supplies
- Waxing supplies
- Hair supplies later

Rules:
- Do not auto-buy.
- Do not scrape sites illegally.
- Use vendor-entered links first.
- Support affiliate links later but do not require now.

Acceptance tests:
- Vendor can view supply options
- Vendor can save supplier
- Vendor can mark ordered
- Inventory restock transaction is logged
